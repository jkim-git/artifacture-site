Artifacture is a free, easy-to-use desktop program for interacting with AI language models. Simply supply your API keys and choose from 20+ leading language models. [Detailed Guide]

Features #

  • Branching
  • Hyperparameter tuning
  • Tokenizer selection and context window size specification
  • Context curation with template messages
  • In-context configuration updates and annotation
  • And more
